# backend is scaffolded with [Initex](https://www.npmjs.com/package/initex).

## How to run?

1. Install dependencies:

   ```bash
   bun install
   ```
2. Setup the database:

   ```bash
   bun run db:up
   ```

3. Run the project:

   ```bash
   bun run dev
   ```

4. Open http://localhost: in your browser to view the project.
